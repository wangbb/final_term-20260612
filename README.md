# 透明方塊互動漸層

作者：wangbb

這是一個使用 p5.js 製作的互動網站作品。畫面以二維陣列產生透明方塊，讓方塊佈滿整個螢幕，並搭配漸層背景與滑鼠互動產生動態視覺效果。

## 檔案結構

- `index.html`：網站結構、landing page、p5.js 載入
- `style.css`：頁面版面、landing page 與互動畫面樣式
- `script.js`：p5.js 畫布、二維陣列方塊與互動邏輯

## 互動方式

- 滑鼠 X 軸：控制畫布中方塊的大小變化
- 滑鼠 Y 軸：控制漸層背景與方塊顏色變化

## 使用方式

直接用瀏覽器開啟 `index.html`，點擊「進入作品」即可開始互動。

此作品透過 CDN 載入 p5.js，因此執行時需要網路連線。
